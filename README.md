### Desarrollo

Gestor de Proyectos: Desde la pagina del gobierno https://www.argentina.gob.ar/noticias/hub-tecnoeducativo-7-proyectos-para-potenciar-los-procesos-educativos se extrayeron 3 proyectos de los mas prometedores y de diferentes categorias, se inventaron y buscaron los dos restantes. La pagina contiene etiquetas:
  * header y nav: comunes en toda la pagina para el enlace y la navegacion entre las mismas
  * ul: para listas sin orden
  * li: para elementos de una lista
  * article: para el uso de un elemento independiente en el momento de implementar proyectos
  * section: para separar

Detalle: Para el detalle construi de arriba hacia abajo la pagina, agregando etiquetas de titulo para ayudar con la jerarquia de la pagina. Para la lista y los hipervinculos utilize la etiqueta <a> y le agregue el detalle de target para que los enlaces se habran en una pagina nueva.
